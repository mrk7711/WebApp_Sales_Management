
# Web Sales Management App

A Flutter Web application for managing product serial numbers, sales records, and CSV export.

## Features

- Search serial numbers
- Display product information
- Save sales records
- Date picker support
- Export sales data to CSV
- Responsive UI for desktop and web

## Technologies

- Flutter Web
- PHP API
- MySQL Database
